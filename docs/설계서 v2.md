# 설계서 v2

> **목적**: 물푸레 1호 A액 / B액의 EC·pH 응답곡선 확보 →
> 
> 
> 운용 중 탱크 상태(EC, pH)를 읽어 **A 또는 B 도징량을 역산하는 알고리즘 모델 구축**의 기초 데이터 수집
> 

> **v2 변경사항**: 실험 베이스 용량을 5L → **4L**로 변경. 표준 1배 도징량이 A 5mL + B 5mL로 정수화되어 계산 편의성 향상.
> 

---

## 실험 조건 요약

| 항목 | 조건 |
| --- | --- |
| 사용 배양액 | 물푸레 1호 A액 / B액 |
| 용기 용량 | 5L 용기 |
| **실험 베이스 수량** | **4L** (용기 여유 공간 1L, 약 20%) |
| **표준 1배 도징량** | **A액 5mL + B액 5mL** (20L 기준 25mL의 1/5) |
| 실험수 | 수돗물 (매 실험 동일 수원 사용) |
| EC 미터 정밀도 | ±100 μS/cm (저가형 / 아두이노 연결형) |
| pH 미터 정밀도 | ±0.1 (저가형 / 아두이노 연결형) |
| 주사기 최소 눈금 | 0.1mL |
| 투입 스텝 | **1.0mL 단위** (정밀도 한계 고려) |

> ⚠️ **저가형 미터 주의사항**
> 
> 
> EC ±100μS/cm, pH ±0.1의 정밀도에서 0.1mL 스텝 투입 시 측정값 변화가 오차 범위 이내일 수 있음.
> 
> **1.0mL 스텝**으로 충분한 신호 변화를 확보하는 것을 권장.
> 

> ℹ️ **용량 여유 공간 확인**
> 
> 
> 5L 용기에 4L 충수 시 1L(20%) 여유 공간 확보. 교반 시 액 넘침 없음.
> 
> 표준 1배까지 누적 투입량(A+B = 10mL)은 4,000mL 대비 0.25% 수준으로 부피 오차 무시 가능.
>

>

> 미터 내부 회로 보정이 아니므로 표준액 기종 종속성 없음.

>

## 데이터 분석 방법

### 1. EC 응답 선형 계수 도출

```
각 실험의 EC 기울기 = (표준 1배 EC - 베이스라인 EC) / 5.0mL

실제 결과 (v2 데이터):
  A액 단독: (1252 - 486) / 5.0 = 153.2 μS/cm per mL
  B액 단독: (844  - 460) / 5.0 = 76.8  μS/cm per mL
  A+B 혼합: (1634 - 446) / 5.0 = 237.6 μS/cm per mL
```

### 2. A/B 개별 EC 기여 비율

```
A액 EC 기여율 (%) = (A단독 EC 상승량 / 혼합 EC 상승량) × 100
B액 EC 기여율 (%) = (B단독 EC 상승량 / 혼합 EC 상승량) × 100

실제 결과:
  A액 EC 기여율 = 766 / 1188 × 100 ≈ 64.5%
  B액 EC 기여율 = 384 / 1188 × 100 ≈ 32.3%
```

> 이 비율이 도징 알고리즘에서 A 또는 B 중 어느 쪽이 EC에 더 크게 기여하는지 판별하는 핵심 계수
> 
> 
> A단독(766)+B단독(384)=1150 μS/cm인데 실제 혼합 상승량은 1188 μS/cm로 약 3.3%(+38 μS/cm) 더 큼 → 미세한 양(+)의 비선형 시너지 존재. 도징 알고리즘은 우선 A/B 개별 선형 계수로 근사하고, 이 잔차는 오차 허용범위 내로 간주.
> 

### 3. pH 편향 방향 확인

```
A액 단독: pH 상승 확인 → 석회(Ca) 성분 영향
B액 단독: pH 하강 확인 → 인산(P) 성분 영향
혼합:     pH 안정 구간 확인 → 이 구간이 운용 목표 범위
```

실제 결과:

```
A액 단독: pH 7.85 → 7.63 (표준 1배까지는 7.85 부근 유지, 6~8mL 구간에서 오히려 하강)
          → 예상(상승)과 다름, 재확인 필요
B액 단독: pH 7.96 → 6.51, 초기 투입부터 뚜렷한 하강
          → 예상대로 인산(P) 성분 영향 확인
혼합:     표준 1배(A5mL+B5mL) 지점 pH 6.58
          → 운용 목표 범위 후보값
```

> ⚠️ A액의 "석회 성분 → pH 상승" 가설은 이번 데이터에서 확인되지 않음. 표준 1배(5mL)까지는 pH 변화가 거의 없다가 6~8mL 구간(고농도)에서 하강 전환됨. 석회의 완충 작용이 예상보다 약하거나 질소·칼리 성분의 산성화 영향이 더 클 가능성.
>

## 도징 알고리즘 설계 방향 (실험 후 적용)

> v2 데이터로 도출한 실제 계수. 아래 기울기는 **4L 실험 조건 기준**이며, 실제 운용 탱크(20L, 표준 1배 = A 12.5mL + B 12.5mL 가정 — "실험 조건 요약"의 20L 기준 25mL 참고)에 적용 시 그대로 쓸 수 없음: 같은 ΔEC를 만드는 데 필요한 mL은 부피에 비례하므로, μS/cm per mL 계수를 실제 탱크에 적용하려면 **(실험 부피 / 실제 탱크 부피)** 배로 나눠야 함. 예) 20L 탱크라면 4L/20L = 1/5 → A액 계수는 153.2 → 30.6 μS/cm per mL로 환산.
> 
> - A액 기울기: 153.2 μS/cm per mL (4L 기준)
> - B액 기울기: 76.8 μS/cm per mL (4L 기준)
> - 혼합 기울기: 237.6 μS/cm per mL (4L 기준)
> - A/B EC 기여율: A 64.5% / B 32.3%
> 

```
[입력] 현재 탱크 EC, pH 측정값
        ↓
[판단 1] EC가 목표값보다 낮은가?
        YES → 도징 필요
        NO  → 대기
        ↓
[판단 2] pH 보정 (실측 반영: A는 pH 중립·EC 전용, B만 pH를 낮춤)
        pH가 목표보다 높은가?
        → B 비중을 늘려 pH를 끌어내림 (B가 EC도 함께 올림)
        pH가 목표보다 낮은가?
        → 양액으로는 pH 상향 불가. A(중립)로 EC만 맞추고,
          필요 시 pH-up(염기)으로 별도 조치
        pH가 목표 범위 내인가?
        → EC 목표에 맞춰 A·B 배분 (pH 유지)
        ↓
        ※ 실제 계산은 위 heuristic이 아니라 EC·pH를 함께 푸는
          연립해(dosing.py)로 수행됨. 위 흐름은 개념적 직관.
        ↓
[계산] 목표 EC까지 도달하는 도징량
        = (목표 EC - 현재 EC) / EC 기울기 계수
        (실제 탱크 부피로 환산된 계수 사용 — 혼합 237.6, A 153.2, B 76.8 μS/cm per mL는 4L 기준값)
        ↓
[출력] 도징량 (mL)
```

## 주의사항 요약

1. **주사기 전용 사용**: A액 주사기와 B액 주사기 혼용 절대 금지
2. **투입 순서**: 항상 A 먼저 → 교반 → B 투입
3. **안정화 대기**: 교반 후 2분 대기 후 측정 (pH는 특히 중요)
4. **센서 헹굼**: 매 측정 전 정수로 전극 헹굼
5. **동일 수원**: 모든 실험은 같은 날 받은 수돗물 사용 권장
6. **베이스라인 기록**: 매 실험 시작 시 반드시 수돗물 EC·pH 기록
7. **과잉 구간 측정**: 표준 1배 이상도 3스텝 반드시 측정 (비선형 구간 파악)
8. **교반 도구**: 유리막대 사용 (금속 도구는 이온 용출로 EC 오차 유발)
